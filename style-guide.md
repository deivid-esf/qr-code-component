# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 423px and 720px
- Desktop: 1024px

## Colors

- White: hsl(0, 0%, 100%)
- Light gray: hsl(212, 45%, 89%)


## Typography

### Body Copy

- Font size (h3): 20px;
- Font size (paragraph): 16px;

### Font

- Family: sans serif;
- Weights: 600.
